# **How do I create a Client Account?**

This guide provides step-by-step instructions on setting up a client account in the Agency panel.

**Step 1:** Navigate to the \*\*Agency Panel \*\*and **Add a New Client**

\*\*Step 2: \*\*Enter a **Name** for the New Client and select a **Voice Provider** (e.g., VAPI or RETELL). 

\*\*Step 3: \*\*Enter API Keys: Provide the **Private Key.**

![Jq7ccwu7qs2ej HL13B Wc Mjfjvt A4 NE 4A Pn](/images/Jq7ccwu7qs2ejHL13BWcMjfjvtA4-NE_4A.png)

**Step 4:** Sync Resources like **AI Agents** and **Knowledgebase**.

![Bp Hi OUX Hpc U Da Sc6i T Rm5h Hop7c O7u EQ Pn](/images/-bpHiOUXHpcUDaSC6iT_RM5hHop7cO7uEQ.png)

**Step 5:** Under Trial Settings, Set **Trial Days** and **Trial Minutes.**

\*\*Step 6: \*\*Tap ****complete. Now, yourClient account is ready.****
